Metronic by default uses Fontawesome icons in the sidebar, header menus, dropdowns, portlets and more.
This page demonstrates how you can use Simple Line Icons icons in the sidebar and header menus.

Simple Line Icons. 162 Beautifully Crafted Webfont Icons.
For more info check out http://graphicburger.com/simple-line-icons-webfont/

NOTE: Icon will be served via the specified class name. To use Simple Line Icons just include assets/global/plugins/simple-line-icons/simple-line-icons.min.css right after the fontawesome css file.
